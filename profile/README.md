## 🚀 项目简介

**XoreIM** 是一个使用 **Rust + Tokio** 编写的高性能即时通讯框架，旨在为开发者提供一套稳定、安全且易于扩展的 **IM 服务端解决方案**。  
相比传统的 Go 或 Java 实现，XoreIM 充分利用了 Rust 的 **零开销抽象** 与 **内存安全特性**，在性能与可靠性之间取得了极佳的平衡。

XoreIM 适合用于：
- 📱 聊天应用（单聊 / 群聊）
- 💬 企业通信平台
- 🧩 游戏内聊天系统
- 🌐 IoT 消息网关
